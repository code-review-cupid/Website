Product Aims & Goals

Finding code reviewers is difficult. You need a sustainable, practical solution!

Built by RSEs, for RSEs and coding researchers, we've created Code Review Cupid - a matchmaking service which will help you find the perfect person to review your code, and let you review theirs in exchange! Everyone has something to contribute, and together we can all write better code.

Code Review Cupid can help connect you with a code review buddy - share skills you're confident in with those who need them, and they'll help you with the skills you need! Reciprocity is our goal, and we want to create code review matches that will benefit both parties!
